### Package 8: Advanced Data Structures
---
*Focus: Pointers, Lists, Trees, and Graphs.*
* Pointer Basics: Declare an integer pointer and display the address it points to.
* Dynamic Allocation: Allocate memory for an array using malloc.
* Pointer Arithmetic: Navigate through an array using only pointers.
* Linked List Node: Create the structure for a single linked list node.
* List Insertion: Implement a function to insert an element at the start of a list.
* List Search: Search for a value in a linked list.
* Doubly Linked List: Create a structure for a doubly linked list node.
* Stack (LIFO): Implement a stack using an array or a list.
* Queue (FIFO): Implement a queue with enqueue and dequeue operations.
* Binary Tree Node: Create a structure for a binary tree node.
* In-order Traversal: Implement recursive in-order traversal of a tree.
* Binary Search Tree (BST): Insert values into a BST according to the rules (lesser left, greater right).
* Graph Representation: Represent a graph using an adjacency matrix.
* Breadth-First Search (BFS): Implement a BFS algorithm for a graph.
* Dijkstra Algorithm: (Simplified logic) Calculate the shortest path between two nodes
